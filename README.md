# Pokedex
This is an express application for displaying a Pokedex.

## Specifications
Node v23.0.0

## Dependencies
* express     ^4.21.1
* nodemon     ^3.1.7

To install dependencies use
```console
npm install
```

## Commands
Command to show LAN IPV4
```console
npm ip
```

Command to start server with nodemon
```console
npm run server
```
